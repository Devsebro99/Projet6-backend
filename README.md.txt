		Pour Installer coorectement le projet N6



1) Creer un dossier " projet N6 ".

2) Copier le frontend du projet suivante https://github.com/OpenClassrooms-Student-Center/Web-Developer-P6 avec " git clone 'adresse ci-dessus' " 
dans le dossier " projet N6 ". Rernommer le fichier "Web-Developer-P6" en " frontend ".

3) Copier le fichier Zip dans le dossier " projet N6 " et faire " extraire ici " pour le deZipper.

4) Donc on a un dossier " projet N6 " avec a l'interieur deux dossiers " frontend " et " backend ".

5) Maintenant on peut faire fonctionner le projet. 
Pour cela , il faut se rendre dans la console :

* Tout d'abord, se rendre dans le dossier " frontend " avec la console (commande: cd frontend ) puis faire " npm install " pour installer toutes les dependances necessaire au fonctionnement du frontend et faire " npm start ". 
	
*Pour voir l'interface du projet, il faut ouvrir un navigateur internet et taper dans la barre d'adresse 
" http://localhost:4200/ ".

*Ensuite, on doit faire fonctionner le backend. 
Pour cela, on doit ouvrir une nouvelle console. 
Dans cette console, on entre dans le dossier " backend " (commande: cd backend ) puis il faut entrer la commande " nodemon server " pour lancer notre API.

6) Voila le projet peut fonctionner correctement.
